Project 1

Katipō spider factsheet

Project overview

This is an informal website with facts about the katipō spider which is native to New Zealand. This website was made as a part of a class project to make a factbook on different native new zealand animals. Everyone chose a different native animal and made a small fact page with information about their chosen animal. Because it was a group project I copied my files into a new repository which is why there are only a few commits.

view project

Details

My end users for this task were my classmates so taking that into consideration I broke up the information and put them in seperate boxes to make it easy to read. I made the theme of the page similar to the colours of the katipō spiders. By doing this it made the page feel unique yet still related to my topic. Part of the criteria for this project was to add a lie within the facts to see if classmates will be able to spot it. I kept the lie in the linked version and if you are wondering what it is you can click the go to home page button to see (the button no longer goes to the home page since I only copied my page )
